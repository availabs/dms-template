DESIGN NOTES · GAPS VS. CURRENT

What this design adds over the current implementation.

Added
  • + KPI strip: site count, subdomain count, themes in use, last modified
  • + Type badge (page / datasets / forms / auth) per row
  • + Subdomain pill — distinguishes root vs. named subdomain
  • + Last-modified column (relative time)
  • + Type filter chips in toolbar (show only page / datasets / …)
  • + Theme column — shows which theme each site uses
  • + Icon-only actions (edit · duplicate · delete) — less visual noise
  • + Sidenav type drill-down for fast filtering
Implementation notes
  • → Pattern type comes from pattern_type field on the pattern row
  • → Last-modified from updated_at — already in the row, just hidden
  • → Theme from theme.selectedTheme stored in pattern config
  • → Subdomain pill: subdomain === '*' → "global"
  • → Delete confirmation dialog already exists; style with tessera tokens
