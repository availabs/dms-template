Foundations · theme · v0.1

Tokens.

The foundational vocabulary every other page in this design system inherits from — colour, type, icons, spacing, radius. Read this page first; everything in components.html and pages/*.html composes these tokens.

01 — Colour

Near-monochrome.

The system is intentionally chromatic-free except for one functional red used only on live-broadcast signals. Greyscale tokens carry every other role — surface, ink, hairline, accent.

02 — Type

Italic display, mono chrome, sans body.

Instrument Serif italic for every headline; Geist Mono uppercase for eyebrows and chrome; Geist for body and form fields. The discipline is the brand — substitute another italic serif at your peril.

Folded in as modifiers, not new tokens: bold weights (700) appear in `textSettings.text*Bold` for occasional emphasis but share the same size + family as their roman parent; size variants between two steps (e.g. a 32px headline) snap to the nearer token rather than creating a new one. The earn-a-token rule (≥ 2px, weight step, family change, role change) is strictly enforced.

05 — Families

Three voices.

--font-display
Aa
Instrument Serif · italic 400

For every headline. The brand's distinctive editorial voice.

--font-sans
Aa
Geist · 300 / 400 / 500 / 600 / 700

For body, captions, form fields, metadata. The default voice for everything that isn't headline or chrome.

--font-mono
Aa
Geist Mono · 400 / 500

For eyebrows, labels, table headers, on-air pills, chrome. The second-most-distinctive voice after italic display.

Loaded via Google Fonts CDN — see _shared.css's @import at the top. Production should self-host these via the theme's fonts array.
