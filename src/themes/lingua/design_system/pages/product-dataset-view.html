Schema

Schema — the row shape, declared.

Every row in this dataset is a typed object. The schema is the contract the rest of the site reads against — any page, chart, or map that consumes subscribers sees the columns below.

Columns

8 columns · 2 required · primary key: id
Column Type Required Default Description
id text · pk ✓ auto Stable subscriber identifier. Format: sub_NNNNN.
name text ✓ — Subscriber's display name as entered at signup. UTF-8.
email text · unique ✓ — Validated email address. Unique across the dataset.
plan enum(4) ✓ "monthly" One of monthly, yearly, founding, comp.
joined date ✓ now() ISO date the subscription started. Used for cohort grouping.
last_open date · null × null Most recent email-open or web-read event. Updated by the open-tracker pipeline.
articles_read int × 0 Lifetime count of unique essays read. Recomputed nightly.
status enum(3) × "active" Derived state: active / lapsed / bounced.

Same shape, expressed as the row format Lingua actually stores. Any page that reads subscribers sees this declaration; any plugin can compose against it.

the-drift/sources/subscribers.schema.js 
// every row, everywhere, has the same shape
export const subscribers = {
  app:      "the-drift",
  type:     "the_drift_env|subscribers:source",
  parent:   "the_drift_env:dmsenv",
  data: {
    title:       "Subscribers",
    description: "All-time subscribers across hosted and exported tiers.",
    primaryKey:  "id",
    columns: [
      { name: "id",            type: "text",  required: true,  default: "auto" },
      { name: "name",          type: "text",  required: true },
      { name: "email",         type: "text",  required: true,  unique: true },
      { name: "plan",          type: "enum",  required: true,
        values: ["monthly", "yearly", "founding", "comp"],
        default: "monthly" },
      { name: "joined",        type: "date",  required: true,  default: "now()" },
      { name: "last_open",     type: "date",  required: false, nullable: true },
      { name: "articles_read", type: "int",   required: false, default: 0 },
      { name: "status",        type: "enum",  required: false,
        values: ["active", "lapsed", "bounced"],
        default: "active" },
    ],
  },
};
