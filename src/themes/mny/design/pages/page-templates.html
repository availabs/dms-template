3.0 — Save as Template widget (Settings pane, bottom)
Settings
Hide in navigation
Sidebar
Save as Page Template
Saves this page's current layout for reuse.
Notes
  • Appears at the bottom of the Settings pane, below all other settings.
  • Saving strips section IDs and refs so sections don't conflict when applied to a new page.
  • Name field: if a template with this name already exists in the pattern, the button changes to Overwrite with a two-step confirm (matching component template behaviour).
  • Success/error shown inline below the button (no toast).
  • Description is optional — shown in the picker and in the manager tab.
