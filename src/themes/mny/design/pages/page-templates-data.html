  • Richtext section · full width · author writes heading + intro

  • Graph section · ½ width · horizontal BarGraph · action_category_json (X-axis), count (Y-axis) · uses fullDataLoad: true on Graph section

status_badge column type: Set column type to status_badge in the Spreadsheet column settings. Covers both implementation_status and action_status values.
