Design System — Patterns

Composed Patterns

Multi-primitive assemblies that recur across MitigateNY pages. Each pattern names the DMS section types, LayoutGroup variant, and component tokens involved so the same composition can be reproduced reliably on any page.

Pattern 01 — Page Header + Action Pair

The recurring top-of-page pattern: eyebrow breadcrumb, H1 with yellow underline, lead paragraph, and a right-anchored action row (Browse All + Export). Lives in a header LayoutGroup variant with clearCentered background.

Home / Hazards & Disasters / Hurricanes
Hazard Profile

Hurricanes & Tropical Storms

Tropical cyclones affecting New York State, including direct landfalls, nor'easter-hybrid events, and inland flooding caused by remnant systems.

DMS sections: Lexical (eyebrow + H1 + lead) + sectionArray col-12 for the action row, or a single Card section with a chrome column type for the CTA buttons.

LayoutGroup: header variant — clearCentered background, topo texture, mr-auto wrapper, pt-[118px] top offset from floating nav.

Pattern 05 — Form Layout (FieldSet + Submit/Cancel)

Standard admin/auth form. FieldSets group related inputs. Submit/Cancel pair at the bottom right. Lives in a content LayoutGroup with max-w-[640px] column constraint.

DMS composition: DMS forms pattern. FieldSets map to logical section groups within the form pattern. Each field is a registered form input type. Submit/Cancel buttons are rendered by the form pattern's action bar. The yellow Save button maps to theme.button.primary; Cancel to theme.button.secondary.

Pattern 06 — Auth / Sign-In Page

Full-screen centred card on topo-textured canvas. Uses the auth Layout variant (no TopNav, no SideNav) with the auth LayoutGroup variant (mx-auto, vertically centred).

Pattern 07 — Section State Indicators

How each section type communicates empty, loading, and error states to the user. These are theme-overrideable via theme.statusIndicators.

Token references: Empty: icon #6D96AE (blue-400), bg #F3F8F9 (blue-50). Loading: skeleton bars #E0EBF0 (blue-100) with CSS animate-pulse. Error: border + icon #DD524C (red-500), bg tint #FCF0F0, text #AA2E26 (red-700).

Pattern 08 — Section Edit-Mode Toolbar (Admin Chrome)

The overlay chrome that appears on a section when an admin is in edit mode. A top bar identifies the section type; a bottom-right action cluster handles move/delete/settings. The section itself shows an edit-mode tint ring.

Pattern 09 — Map Workbench Layout

Full-height map surface with a floating control panel on the left. Uses the app Layout variant (with sidenav) and a clearCentered LayoutGroup. The map canvas is fullwidth; the panel is an absolutely-positioned white card.
