Rich content

Lexical — the rich-text surface

On placing

A page is a row. A chart is a row. A map is a row. When the representation is one thing, the renderings can be many — the same claims table becomes a county dashboard, a printed appendix, and an open data endpoint without being copied once.

The course sets the figure

Authors place sections the way a workshop set tesserae: one at a time, against a fixed grid, each earning its position.

  • Place the data — bind the section to a view.
  • Set the layout — spans on the 12-column grid.
  • Publish — the mosaic is the composition, not the tile.

“Our pages and our datasets used to drift. Now they’re the same row, read two ways.”
