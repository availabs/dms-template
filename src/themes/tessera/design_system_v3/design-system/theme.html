Foundation · tokens

Theme — the brand’s vocabulary

Everything the rest of the system inherits from: twelve mineral colors, sixteen type tokens, a drawn icon set, the grout scale, one shadow, and the small set of ornaments the brand permits itself. If a token is missing here, it is missing everywhere.

Color

Twelve minerals, three duties

Surface is the floor — roughly 85% of any composition. Ink writes on it. Accent is the inlay: at most 15% of any surface, always doing work, never decoration.

Contrast — measured, not assumed

WCAG 2.1 ratios for every sanctioned ink-on-surface pair. Pairs below 4.5:1 are restricted to large type (≥ 24px, or 18.5px medium) or non-text use; fog never carries copy.

Type

Sixteen tokens, four voices

Serif display for what the brand says, humanist sans for what it explains, mono for what it measures. Every visible piece of text in this deliverable resolves to one of these sixteen tokens — color, tabular numerals, italic-on-prose, uppercase tracking, and surface are modifier axes applied at the call site, never new tokens. Icons

Drafted, not drawn

A custom set on a 24px grid: 1.5px stroke, butt caps, mitre joins, square corners — and a rotated square (a tessera) wherever another set would use a dot. Names are the API: components reference theme.Icons[name]; an unregistered name renders nothing. Source of truth is theme/icons.js; audit with icons-audit.mjs --brand tessera.

Stroke stays 1.5px at every size — smaller icons read lighter, which suits the drafted hand. Below 16px, use the Tessera diamond alone.

Rules of the hand
  • Butt caps, mitre joins — terminals squared, never rounded.
  • A rotated square where other sets use a dot: pupils, pins, points.
  • Name the glyph, not the use site: Map, not GeographyTab.
  • Every page svg is tagged <!-- icon: Name --> or <!-- decorative -->; the audit gates the design→theme hand-off.
  • Map and data glyphs follow cartographic legend conventions, not app-store pictograms.

Motion

Snap, don’t ease

Motion is reserved for direct action and meaningful state change. A new element appears the way a tile is set down: a 4px settle, opacity in, scale 0.98 → 1. Reveals follow andamento — the flow of the course, row by row, never all at once.

Token Value Use
dur-snap100mstile placement, hover, press, toggle
dur-settle180mspanel moves, drawers, dialog entry
ease-snapcubic-bezier(.2, 0, .1, 1)the only curve — fast in, firm stop
andamento stagger18–30ms / tilemulti-element reveals settle course by course

Forbidden: parallax, looping background animation, scroll-triggered drama, any ease longer than 180ms. prefers-reduced-motion lands everything already placed.

The press — hover, then set

Active state moves down 1px and drops the lift — the tile is set. 100ms, snap curve, nothing whooshes.

Ornament

The floor is mostly stone

The reference is a Cosmati pavement, not a mosaic backsplash: ~85% plain surface, ~15% considered inlay. The brand permits itself exactly four ornaments, each with a sanctioned place. Anything else is decoration, and decoration is refused.

Friezes — band boundaries only
tessera course · .tsr-frieze-tiles
running meander · .tsr-frieze-meander

Sanctioned at three places: the base of a marketing hero, the top of a footer, and long-form section breaks. One frieze per boundary; never stacked, never tiled as a background.

Tabula ansata — the designator

The handled tablet Roman workshops carved their names into. One per band, on marketing surfaces only; product chrome uses the plain metaSM eyebrow.

The mark — a tessera, set
Tessera

A single tile set within its frame, rotated 45° — placed, not stamped. Monochrome at 16px for the favicon; ornament belongs to the marketing site, never to the mark.

Emblema — one featured panel
Featured dataset
NFIP claims, 1978–2026

The finely-worked central panel set into a plainer floor: a double frame (1px slate outside, 1px grout inside) marks at most one element per surface as the emblema. Two emblemata cancel each other.
