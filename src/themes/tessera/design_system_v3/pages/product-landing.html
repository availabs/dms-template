The craft

Place. Set. Publish.

Place the data

A tessera is a single typed row. Upload a spreadsheet, point at a Postgres table, or write rows by hand — each lands as a row in data_items, typed and addressable.

Set the layout

Sections are set on a 12-column bed, course by course. Pick a span, choose the columns, bind the source — the same grid carries a homepage and a dataset workbench.

Publish the mosaic

The site is what the rows compose. Change a row and every rendering that reads it — page, chart, map, feed — is already current. Nothing else to deploy.
