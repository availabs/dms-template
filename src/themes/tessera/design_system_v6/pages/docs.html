quick start CLI reference

quick start

Three commands to a live page

01 · create

Scaffold a site

One command creates a site with a home page and opens the editor in the browser.

$ npx tessera new my-site

02 · add data

Upload a dataset

A CSV or shapefile becomes a versioned dataset. In the editor, drop a table or chart section and bind it.

$ tessera dataset upload claims.csv

03 · publish

Put it live

Publishing snapshots the draft. Visitors see the snapshot; your next draft starts from it.

$ tessera publish

Prefer the browser? Everything above has a button — the CLI is optional end to end.
