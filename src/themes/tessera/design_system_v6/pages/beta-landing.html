features

What tessera does today

It's not a prototype — three production sites already run on it. Here's the tour.

edit in place

The page is the editor. Click a section and type — layout, theme, and live data update as you work, so the page you're editing is exactly the page visitors will see.

a grid you can feel

Every page is sections on a 12-column sheet. Drag them, span them, stack them. Toggle the grid overlay to see exactly where everything sits.

your data, live

Upload a CSV or a shapefile and it becomes a real dataset with versions. Cards, tables, and charts bind to it — update the data, every page updates.

tables built for real data

Sort, filter, paginate, search across sixteen columns at once. A keyword search over 17k rows is a config option, not custom work.

charts without custom code

Pick a dataset, pick an x and a y, get a chart in the site's palette. Target lines are just a second series — no bespoke code required.

maps from real GIS data

GIS data renders as real vector maps — parcels, corridors, county lines. A land bank runs its whole property viewer on this.

themes all the way down

Every button, nav, and card reads from a theme. Swap it and the same content becomes a different site. This page is a theme. So is the dark mode you can toggle up top.

how it works

Place, bind, publish

01 · place

Drag a section onto the sheet

Text, card, table, graph, map — pick one, drop it, give it columns. The grid does the layout math so you don't.

02 · bind

Point it at a dataset

Choose the source, choose the columns, choose how each one renders — image, link, formatted number. The section fills itself.

03 · publish

Publish when you're ready

Drafts stay drafts until you publish. Publishing puts the page live; visitors never see work in progress. Undo exists.

faq

Frequently asked questions

what does the beta cost?

Nothing — the beta is free. If paid plans ever arrive, everyone from the beta keeps a free plan.

what if the project ends?

Your content is yours — pages and datasets export as JSON and CSV whenever you want.

what's it built on?

React and Postgres, with a data layer built for maps and charts. There's a CLI too — tessera page create — if you prefer the terminal.

can an update break my live site?

Published pages are immutable snapshots — an update can break the editor (see known issues), but never a live page.

Bring a project.

A field log, an archive, a dashboard over a live dataset — the beta works best with a real use in mind. Request an invite and build it on real data.
