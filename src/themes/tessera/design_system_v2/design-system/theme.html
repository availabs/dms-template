design system / theme

Theme — the foundational tokens everything else inherits.

Color, type, icons, spacing, radii, and elevation. The vocabulary the brand speaks before it composes anything. Every primitive in components and every pattern in patterns reads from this page.

SECTION · COLORS

Colors — mineral palette, three roles.

~85% surface, ~15% accent. Surface is the floor; accent is the inlay. Pure white, pure black, tech-blue, neon, and gradients are out of vocabulary. Accents appear as emphasis on labels, links, and warnings — never as decoration.

SECTION · TYPE

Type — two families, four roles.

Newsreader (display, in place of Tiempos Headline until licensed) does the headlines and long-form. IBM Plex Sans is the UI and body voice. IBM Plex Mono renders every number, code sample, and tabular surface. Tabular figures everywhere a digit appears.

A tessera is a single tile in a Roman or Byzantine mosaic — the smallest unit of a deliberately composed surface. Roman and Byzantine craftsmen spent careers placing tesserae one at a time into floors and ceilings that lasted millennia. Each tessera is small and ordinary; the composition is what's beautiful and durable.

SECTION · ICONS

Icons — drafted, not drawn.

Geometric, line-based, 1.5px stroke, no fills. Lucide as the working set; swappable for Phosphor or a custom drawn set without re-keying. Icon names are the API — components reference them by name through the theme's Icons map.

SECTION · SPACING & RADII

Spacing & radii — generous space, square corners.

4px base. Tiles have square corners; the only rounded affordances are buttons, inputs (2px), and pills (full). If a layout feels tight, double the margin before reaching for a smaller font.

SECTION · ELEVATION & MOTION

Elevation & motion — flat by default, snap when moved.

One lifted treatment; no glass, no frosted blur, no large drop shadows. Animations snap into place (80–200ms cubic-bezier(0.2, 0, 0.1, 1)) — nothing whooshes, nothing fades for long.

Elevation

flat · the default
no shadow · 1px grout-light frame
lifted · the single elevation
--shadow-lift · 0 1px 2px rgba(42, 47, 54, 0.04)

Motion

--dur-snap · 100ms · tile placement, hover
--dur-move · 180ms · larger transitions
--ease-snap · cubic-bezier(0.2, 0, 0.1, 1)
Reveals are 4px baseline-shift + opacity 0→1 + 0.98→1 scale at most. No parallax. No looping background animation. No 400ms crossfades.
