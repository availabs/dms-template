// 02 · who's it for

For state planners. For MPOs. For the public.

TransportNY started as an NYSDOT internal reporting tool and grew into a shared platform. The MPOs use it to draft their TIPs. NYSDOT uses it to file MAP-21 PM3 federal reports. Academics pull the same data for research. Journalists check the same dashboards before running a story. Public-read access requires no account.

  • NYSDOT — REGIONS 1–11
    Federal reporting, internal TIP analysis, congestion mitigation.
  • METROPOLITAN PLANNING ORGS
    All 14 NY MPOs draft and publish their TIP / UPWP reports here.
  • ACADEMIC + PUBLIC
    Free public-read access. CSV export, REST + GraphQL APIs, GIS layers.
