// 02 · STRUCTURE

Grid.

DMS pages are nested: Site → Layout → LayoutGroup → Section → Component. Layouts own page chrome; LayoutGroups are bands within the page; Sections render inside groups and host one component each.

01

The five-layer hierarchy

Site · DmsSite · supplies theme via ThemeContext
└── Layout · outer wrapper · optional TopNav · optional SideNav
└── LayoutGroup · a band of content (header / content / footer)
└── Section · one editable block · wraps one component
└── Component · Card · Spreadsheet · Graph · Map · Lexical · …
└── Primitives · Button · Input · MultiSelect · Pill · Icon · …
